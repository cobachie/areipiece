# Areipiece

Areiの基本的なピースを登録します.  
  
- ヘッダー  
- フッター  
- ページタイトル
- パンくずリスト
- アクセシビリティツール


## Installation

Add this line to your application's Gemfile:

    gem 'areipiece'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install areipiece

## Usage

    $ areipiece piece
